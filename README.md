# Computer Science Projects Portfolio (C++) - Justyna Miazek

This repository contains various implementations of advanced data structures, optimization algorithms, and low-level C++ programming concepts developed during my studies at the University of Łódź.

## Key Projects (Recruiter Highlights)

### 💎 [Segment Tree Implementation](projekt4_Drzewa_przedzialowe_Justyna_Miazek.cpp)
An efficient implementation of a Segment Tree data structure for range queries and point updates. Ideal for performance-critical systems like GIS or real-time data processing.
* **Complexity**: Build $O(n)$, Query $O(\log n)$, Update $O(\log n)$.

### 💎 [Open Addressing Hash Table (Dictionary)](slownik_adresowanie_otwarte_Justyna_Miazek.cpp)
A high-performance implementation of a Dictionary/Hash Table using **Open Addressing** for collision resolution. Focused on optimizing load factor and memory management.
* **Efficiency**: Achieves $O(1)$ average time complexity for operations.
